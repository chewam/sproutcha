SproutCore vs Sencha vs whatever...

The aim of this project is to compare the libraries SproutCore and Sencha. This comparison is done on small examples illustrating some imporant features of these two frameworks. The examples are not intended to teach the best practices of those libraries, but rather to show how to perform a specific task in the easiest way.

Because SproutCore 2 does not have any UI at the moment, the graphical components of Sencha are not used and all examples are based on the CSS library Bootsrap, from Twitter.

Note that the comparison is not exclusively for SproutCore and Sencha, feel free to provide examples based on other libraries on GitHub.

Model load and update data

In this example a model is created, data is loaded from the server and updated thanks to a form.

  • ExtJS 4
  • SproutCore 2
example

Define the model

Ext.define('Employee', {
    extend: 'Ext.data.Model',
    proxy: {
        type: 'rest',
        reader: {root: 'objects'},
        url: 'http://demo.revolunet.com:9876/api/v1/employee'
    },
    fields: ['id', 'first_name', 'last_name', 'salary']
});

Load an employee with id = 1

Employee.load(1, {
    success: function (employee) {...}
});

Update and save employee data

employee.set(data);
employee.save({
    success: function (employee) {...}
});
example

Define the store

App = SC.Application.create({
  store: SC.Store.create().from('App.DataSource')
});

Define the data source

App.DataSource = SC.DataSource.extend({
  retrieveRecord: function(store, storeKey, id) {...},
  updateRecord: function(store, storeKey) {...}
});

Define the model

App.Employee = SC.Record.extend({
  primaryKey: 'id',

  firstName: SC.Record.attr(String),
  lastName: SC.Record.attr(String),
  salary: SC.Record.attr(Number)
});

Load an employee with id = 1

App.set('currentEmployee', App.store.find(App.Employee, 1));

Update data via bindings

{{#view App.Form contentBinding="App.currentEmployee"}}
  {{view App.TextField valueBinding="content.first_name"}}
{{/view}}

Save employee data

App.store.commitRecords();

Store browse a set of data

In this example a set of elements is loaded from the server. Elements are paged 10 by 10. Templates are used to display the list of elements and information about the pagination.

  • ExtJS 4
  • SproutCore 2
example

Define the model

Ext.define('Employee', {
    extend: 'Ext.data.Model',
    proxy: {
        type: 'rest',
        reader: {
            root: 'objects',
            totalProperty: 'meta.total_count'
        },
        startParam: 'offset',
        url: 'http://demo.revolunet.com:9876/api/v1/employee'
    },
    fields: ['id', 'first_name', 'last_name', 'salary', 'company']
});

Create and load store

var store = Ext.create('Ext.data.Store', {
    autoLoad: true,
    pageSize: 10,
    model: 'Employee'
});

Render employee list

gridTpl.overwrite('grid', store.getRange());

Change store page

store.nextPage();
store.previousPage();
